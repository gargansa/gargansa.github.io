Projects

  • Created a single player word game called Hexwordz. Designed with GameMaker Studio. The game generates a set of letters based on the percentage that those letters exist in the dictionary. This prevents a frequency of Q's and Z's from appearing. The base dictionary was created from "The ENABLE dictionary" Enhanced North American Benchmark Lexicon. Additional words were added through a simple script i wrote that would scan public domain books. It would add new words to a seperate list if they were not already in the dictionary, the smaller list was easier to review and combine. The game also does a check to see if any words remain on the board. This is handled by a recursive method that checks each tile to see if a neighbor creates a 2 letter word. If it gets through all the tiles without a solution it then checks for permutations of 3 letter words and then 4 letter and onward until it runs out of tiles.

  • Led developement to build a python plugin for an opensource software known as Cura. The project enabled 3d printers such as the Promega and Crane made by M3D, mix plastics within the nozzle at varying rates. The results create gradients and other effects through the height of the print.

    Code can be found on my github

    MELT Prints

  • Created a recursive tree generator. The code works by recursivly calling a createBranch function 2-3 times. Each time reducing the size of the branch which doubles as an exit strategy for the recursive call, and gives the tree the effect of smaller branches towards the tips. The colors are randomly generated initially from a limited range, restraining them from very dark and very light colors. At the end of each branch a leaf is drawn based on a rotated color where rgb values become gbr values, this ensures the leaves stand out.
